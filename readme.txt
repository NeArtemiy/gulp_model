Основные команды:
    gulp clean - очищает папку dist
    gulp fontsAdd - подключает шритфы в css
    gulp svg_sprite - объединяет все svg картинки в один фаил для оптимизации

css плагины:
    gulp-sourcemaps - используется для отображения исходного кода без сжатия
    gulp-file-include - позволяет добавлять элементы кода в фаил
    gulp-sass - преобразует sass в css
    gulp-group-css-media-queries - оптимизирует все медиа запросы и добавляет их в конец кода
    autoprefixer - добавляет префексы в css для оптимизации старых версий браузеров
    gulp-webp-css - при добавлении изображений в css так же создает вариант добвления и для webp
    clean__css - оптимизирует код css

images
    webp - конвектирует изображение в webp
    gulp-imagemin - компрессия изображений

fonts
    ttfwoff2 - конвектирует ttf в woff2
    ttfwoff - конвектирует ttf в woff
    svg_sprite - все svg элементы добавляет в один файл для оптимизации
    также автоматически подключает их в проект
